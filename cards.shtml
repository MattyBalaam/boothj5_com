Playing Cards

I enjoy learning about different programming languages and paradigms. To get to grips with a language, I usually write the card game shithead with it.

The images link to the source code on Github, I'm always keen to help people learn about the same things, and also like to hear from those who can help me learn to write better code in a certain langauge.
